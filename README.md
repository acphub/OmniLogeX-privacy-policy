# 個人情報保護方針

**株式会社ACP**

制定日：2026年5月1日
バージョン：Ver.1.0

株式会社ACP（以下「当社」）は、OmniLogeX サービス（以下「本サービス」）の運営にあたり、ユーザーの個人情報の保護を重要な責務と認識し、以下の方針に基づき適切に取り扱います。

---

## 第1条（収集する情報の種類）

本サービスの提供にあたり、以下の情報を収集します。なお、以下に列挙する情報のうち、特定の個人を識別できるもの、または他の情報と容易に照合することで個人を識別できるものが、個人情報保護法上の「個人情報」に該当します。

### （1）契約・利用登録情報

* 法人名・事業所名・担当者氏名
* メールアドレス・電話番号
* テナントID・サイトID・利用契約に関する情報

### （2）利用者情報

* 氏名・ユーザーID
* ログイン認証情報（パスワード等）
* 勤務情報（出退勤日時・メモ等）
* アルコール検査情報（検査日時・呼気アルコール濃度数値・判定結果（合否）・再検査情報・使用センサーID・検査回数）
* 休憩情報（休憩開始・終了日時）

### （3）端末・技術情報

* 端末シリアル番号（device_sn）
* デバイストークン（アプリ認証用の端末識別子）
* ライセンスキー
* OSバージョン・端末機種情報
* アプリバージョン
* リクエスト相関ID（監査ログ連携・トレーサビリティ確保用）

### （4）位置・センサー情報

* GPSによる位置情報（緯度・経度・測位精度、業務中の定期的・継続的な取得を含む）
* Bluetooth Low Energy（BLE）による呼気検査デバイスの識別情報
* NFC / RFIDタグ読取情報（タグID等）

### （5）業務関連情報

* 車両情報（車両ID・ナンバープレート番号・車両アラート情報（整備期限・点検期限・燃料残量警告）・車両バインドトークン）
* スキャン情報（スキャン日時・ラベルID・品名・数量・スキャン結果（正常／異常／重複）・手動入力フラグ・スキャン地点座標・配送先・停車地点ID）
* 配送情報（配送日・配送先名称・配送先住所・パレットID・品目ID・品目名・必要数量・配送状況）
* 監査ログ（操作履歴・相関IDによるトレース情報）

### （6）問い合わせ情報

* お問い合わせ内容・氏名・メールアドレス等

## 第2条（個人情報の利用目的）

収集した情報は、以下の目的の範囲内でのみ利用します。

* 本サービスの提供・運営・維持管理
* 利用者認証およびデバイス認証・アクセス制御
* 勤怠管理・勤務状況の記録・報告
* アルコール検査記録の管理および法令（道路交通法等）遵守対応
* 配送業務の管理・追跡・最適化
* GPS位置情報に基づく業務記録・安全運転支援
* NFC・RFID技術を用いた物品管理
* バーコードスキャン機能による荷物・物品の管理処理
* 車両管理（車両状態の把握・整備記録）
* 不正利用の検知・セキュリティ対策・監査対応
* 本サービスの改善・新機能開発のための統計分析（個人を識別しない形式に限る）
* ユーザーサポート・お問い合わせ対応
* 法令に基づく開示要求への対応

## 第3条（位置情報の取り扱い）

本サービスでは、業務管理の目的のため、端末のGPS機能を使用して位置情報を取得します。

* **継続的な位置情報の収集**：本サービスは、業務中（勤務開始から退勤まで）の間、定期的に位置情報（緯度・経度・測位精度）を収集・サーバーへ送信します。これは配送ルートの記録、安全運転支援、および労務管理の目的で利用されます。
* **イベント連動の位置情報**：出退勤打刻、アルコール検査、休憩開始・終了、スキャン操作等の各業務イベント発生時にも、そのイベントに紐づけて位置情報を記録します。
* **位置情報の精度**：本サービスはAndroidの高精度GPS位置情報を使用します。
* **バックグラウンド動作**：業務継続追跡のため、アプリがバックグラウンドにある状態でも位置情報を取得する場合があります。
* **位置情報の保存期間**：業務記録として、法令上必要な期間または契約事業者が設定する期間保存します。
* **利用者の権利**：位置情報の収集を端末設定にて拒否した場合、本サービスの一部または全部の機能をご利用いただけない場合があります。

---

## 第4条（アルコール検査情報の取り扱い）

本サービスは、道路交通法および関連法令に基づく安全運転管理者業務の一環として、アルコール検査情報を収集・管理します。

* **収集するデータ**：検査日時、呼気アルコール濃度の数値（mg/L）、判定結果（合否）、使用センサーID、検査回数、再検査の場合はその理由（コンテキスト情報）を収集します。
* **デバイス接続**：アルコール検査デバイスとの接続にBluetooth Low Energy（BLE）を使用し、デバイスの識別および検査データの取得を行います。
* **利用目的**：安全運転の確保、法令遵守の記録・証跡保全、事業者による安全運転管理者業務の遂行。
* **健康関連情報としての取り扱い**：呼気アルコール濃度は健康状態に関連する情報に準じるものとして、特に厳格に管理します。本データへのアクセスは、業務上必要な担当者（安全運転管理者等）に限定します。
* **第三者提供の制限**：法令に基づく行政機関（警察・運輸局等）への報告を除き、事業者の許諾なく第三者に提供しません。

## 第5条（端末機能の利用）

本アプリケーションは、以下の端末機能・ハードウェアを使用します。

### カメラ（ライト機能）

作業環境の照明補助として、カメラのフラッシュライト（トーチ）機能を使用します。カメラによる撮影・画像処理は行いません。

### バーコードスキャナー

荷物・商品のバーコード読み取りに、専用のレーザースキャナーを使用します。スキャンにより取得したラベルIDおよび業務情報は第1条（5）の通りサーバーに送信されます。

### Bluetooth Low Energy（BLE）

アルコール検査デバイスの識別および接続に使用します（第4条参照）。Android 12以降はBLUETOOTH_SCAN・BLUETOOTH_CONNECT権限、Android 11以前はBLUETOOTH・BLUETOOTH_ADMIN権限を使用します。

### NFC / RFID

NFCタグおよびRFIDタグの読み取りによる物品管理に使用します。

### GPS（位置情報）

第3条に記載の通り、業務管理目的で位置情報を収集します（ACCESS_FINE_LOCATION・ACCESS_COARSE_LOCATION権限を使用）。

これらの機能の利用許可を、インストール時またはアプリ起動時にAndroidシステムが求めます。各機能の許可を拒否した場合、関連する機能がご利用いただけない場合があります。

## 第6条（個人情報の第三者提供）

当社は、以下の場合を除き、収集した個人情報を第三者に提供しません。

* ユーザー本人（または法人契約者）の同意がある場合
* 法令に基づく場合（行政機関・捜査機関からの適法な開示要求等）
* 人の生命・身体・財産の保護のために必要であり、本人の同意を得ることが困難な場合
* 公衆衛生の向上または児童の健全な育成のために必要であり、本人の同意を得ることが困難な場合
* 国または地方公共団体が法令の定める事務を遂行することに協力する必要がある場合

なお、本サービスの提供に必要なクラウドインフラ事業者（サーバー運用委託先等）への業務委託は、第三者提供には該当しません。当社は委託先との間で個人情報の適切な保護に関する契約を締結します。

## 第7条（個人情報の安全管理）

当社は、収集した情報の漏洩・滅失・毀損を防止するため、以下の安全管理措置を講じます。

* **通信の暗号化**：すべてのAPI通信はHTTPS（TLS）により暗号化されます。
* **認証・アクセス制御**：Bearer JWTトークンによるユーザー認証、デバイストークンによる端末認証を実施し、不正アクセスを防止します。
* **端末内データの保護**：端末内に保存する認証情報・個人情報は、Android Keystoreを利用した暗号化ストレージに保存します。
* **監査ログ**：すべての操作は相関IDと紐づけた監査ログとして記録・追跡可能な状態で保持されます。
* **アクセス権限管理**：業務上必要な担当者のみが個人情報にアクセスできるよう、ロールベースのアクセス制御を実施します。
* **定期的な安全点検**：セキュリティ脆弱性の点検・対応を定期的に実施します。

## 第8条（データの保存期間）

収集した情報は、以下の基準に従い保存します。

| データ種別           | 保存期間                           |
| --------------- | ------------------------------ |
| 勤怠情報・アルコール検査記録  | 法令の定める期間（労働基準法・道路交通法等に従う）      |
| 位置情報履歴・配送記録     | 契約事業者が設定する期間（デフォルト：最終記録日から1年間） |
| スキャン情報・業務ログ     | 契約事業者が設定する期間                   |
| JWTトークン（ログイン認証） | ログアウトまで                        |
| デバイストークン（端末認証）  | 端末の利用契約終了またはアプリ削除まで            |
| 問い合わせ情報         | 対応完了から3年間                      |

保存期間経過後は、安全な方法で削除または復元不可能な匿名化処理を行います。

## 第9条（ユーザーの権利）

ユーザーは、自己の個人情報について以下の権利を有します。

* **開示請求権**：当社が保有する個人情報の開示を請求できます。
* **訂正・追加・削除請求権**：内容が事実と異なる場合、訂正・追加・削除を請求できます。
* **利用停止・消去請求権**：個人情報の利用停止または消去を請求できます。
* **データポータビリティ**：技術的に可能な範囲で、保有するデータの機械可読形式での提供を請求できます。

これらの権利行使については、第13条の連絡先にご連絡ください。なお、業務記録としての個人情報については、所属事業者（法人契約者）を通じた請求が必要な場合があります。

### Googleプレイストア（Android）をご利用のユーザーへ

アプリ内の設定画面または [support@omnigx.jp](mailto:support@omnigx.jp) へご連絡いただくことで、お客様の個人データ削除申請を受け付けます。削除申請受付後、30日以内に対応いたします。

## 第10条（Googleプレイストア データセーフティ対応）

本サービスのAndroidアプリは、Googleプレイストアのデータセーフティポリシーに基づき、以下の通り収集データを開示します。

### 収集・送信するデータカテゴリ

| カテゴリ      | 具体的なデータ           | 目的           | 必須 |
| --------- | ----------------- | ------------ | -- |
| 位置情報（精細）  | GPS座標（緯度・経度・精度）   | 業務記録・安全運転支援  | 必須 |
| 個人情報      | 氏名、ユーザーID         | 認証・勤怠管理      | 必須 |
| 健康・フィットネス | 呼気アルコール濃度         | 安全運転管理（法令対応） | 必須 |
| デバイスID    | 端末シリアル番号、デバイストークン | 端末認証         | 必須 |
| 業務情報      | 勤怠記録、配送情報、スキャンデータ | 業務管理         | 必須 |

### データの取り扱い方針

* **第三者共有**：法令に基づく場合を除き、第三者とのデータ共有は行いません。
* **暗号化**：転送中・保存中のデータはすべて暗号化されます。
* **削除リクエスト**：ユーザーはデータ削除をリクエストできます。

## 第12条（本方針の変更）

当社は、法令の改正・サービスの変更等に伴い、本方針を変更する場合があります。

* 変更を行った場合は、本サービス上またはアプリ内での通知、および当社Webサイトへの掲載により告知します。
* 重要な変更（収集データ種類の追加、利用目的の変更等）の場合は、メールでの個別通知または再同意の取得を行います。
* 変更後の方針は、本サービス上に掲載した時点から効力を生じるものとします。
